# DP-GRAMS
**Differentially Private Gradient-Ascent Mean-Shift for Mode Seeking**

## Contents
- Modal clustering & regression under \((\varepsilon,\delta)\)-differential privacy
- Private and non-private implementations of MS / PMS
- Synthetic and real-data experiments included
- Reproducible figures and tables
- Supplement to the main paper
